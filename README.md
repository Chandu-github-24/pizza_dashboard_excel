# 🍕 Pizza Sales Dashboard | Microsoft Excel

## 📌 Project Overview

This project is an interactive **Pizza Sales Dashboard** developed using **Microsoft Excel** to analyze sales performance and derive actionable business insights. The dashboard transforms raw transactional data into meaningful visualizations, enabling stakeholders to monitor key performance indicators (KPIs), identify sales trends, and make data-driven decisions.

This project was created as part of my journey to strengthen my data analytics skills and build a practical portfolio while preparing for a **Data Analyst** role.

---

## 🎯 Business Problem

A pizza restaurant generates thousands of sales transactions. However, raw transactional data alone does not provide meaningful insights for business decisions.

The objective of this project is to:

- Monitor overall sales performance
- Identify best-selling and worst-selling pizzas
- Analyze customer purchasing patterns
- Evaluate sales by category and pizza size
- Track sales trends over time
- Build an interactive dashboard for quick business reporting
